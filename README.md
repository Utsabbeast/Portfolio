# Movie Portfolio

A high-end, immersive digital portfolio experience built with a "Noir/Cinematic" aesthetic.

**Live Site:** [https://ubeast.onrender.com](https://ubeast.onrender.com/)

---

## 🎬 Shuriken Production
This project is developed under **Shuriken Production**. It features custom animations, a reactive "Inspect Archive" mode for exploring credentials, and an interactive movie-style credits sequence with branded social interactions.

---

## 🛠️ Material Used
This project leverage several amazing libraries and assets:

- **Core Framework**: [Django](https://wwww.djangoproject.com/) (Python)
- **3D Graphics**: [Three.js](https://threejs.org/) (Using THREE.Fire and GLTFLoader for hand-drawn volumetric effects)
- **Styling**: Vanilla CSS with custom SVG filters (Hand-drawn charcoal noise, film grain, and vignette).
- **Typography**: 
  - `Anton` (Titles/Headers)
  - `Courier Prime` (Archive/Document text)
- **Assets & Credits**:
    - **Hand 3D Model**: [bumstrum on Sketchfab](https://sketchfab.com/bumstrum)
    - **Background Media**: [Freepik](https://www.freepik.com/) (MP4s)
    - **Interactive Icons**: [Simple Icons](https://cdn.simpleicons.org/)
    - **Font Styling Inspiration**: [Uiverse (Little Dingo 13)](https://uiverse.io/AatreyuShau/little-dingo-13)
    - **Environmental Images**: [Stockcake](https://stockcake.com/)
    - **Ambient Soundscapes**: [Pixabay (samuelfjohanns)](https://pixabay.com/users/samuelfjohanns-1207793/)
